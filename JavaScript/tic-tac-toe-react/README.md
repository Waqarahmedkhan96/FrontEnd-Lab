# React Tic-Tac-Toe Practice

This project is based on the official React tutorial:
https://react.dev/learn/tutorial-tic-tac-toe

## What I Practiced

- React component structure
- Functional components
- Props and state
- Event handling in React
- Lifting state up
- Immutable state updates
- Rendering lists with `map()`
- Basic game logic implementation
- React hooks (`useState`)

## Components Implemented

### Game
Controls the overall game logic and stores the game history.

### Board
Displays the tic-tac-toe board and manages square clicks.

### Square
Represents a single square on the board.

## Key React Concepts

- **State management**
- **Component hierarchy**
- **Unidirectional data flow**
- **Reusable components**
- **Game history tracking**

## Technologies Used

- React
- JavaScript
- HTML / JSX
- CSS

## Learning Outcome

This exercise helped me understand how React manages UI updates using state and components while keeping the code modular and reusable.
