# Tic-Tac-Toe Assignment

This folder contains two browser-based Tic-Tac-Toe implementations written in JavaScript:

- `oop.html` → Object-Oriented Programming version
- `fp.html` → Functional Programming version

## Features
- 3x3 Tic-Tac-Toe board
- Status message for current turn, winner, or draw
- Restart button
- Winning line across matched cells
- Fireworks effect on win

## Programming styles
### OOP version
The object-oriented version uses classes to manage the game state and behavior.

### FP version
The functional programming version uses immutable state updates and functions that return new state.
